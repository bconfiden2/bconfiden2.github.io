*# 백준 # 비트마스킹 # 수학*

<br>

[문제 바로가기](https://www.acmicpc.net/problem/1094)

<br>

---

- 문제에서 뭘 장황하게 설명하고 있지만, 결국에는 1, 2, 4, 8, 16, 32 등 2의배수로 수를 만들라는 뜻. 비트만 세주면 된다
- builtin popcount (gcc 내장함수, 정수에서 1 비트 수를 세준다)
- 비트연산자에 친숙해지기

---
<br>

{% highlight c++ %}

#include <iostream>

int n, a;

int main(void)
{
  std::cin >> n;
  // 최대값 64니까 2의 6승까지만 비트마스킹
  for(int i = 0 ; i < 7 ; i++)
  {
    // 해당 자릿수에 비트가 있으면 증가
    if(n & (1 << i))  a++;
  }
  std::cout << a << '\n';
}
