*# 백준 # 소수 판정 # 수학 # 정수론*

<br>

[문제 바로가기](https://www.acmicpc.net/problem/11653)

<br>

---

- 현재 소인수 값을 2부터 시작해서, 현재 소인수로 더 이상 나눠지지 않을 때 까지 나눠준다.
- 현재값에서 전부 나눈 이후엔, 1씩 증가시키면서 나눌 수 있는 소인수가 나올 때 까지 반복한다.

---
<br>

{% highlight python %}

N = int(input())
cur = 2
while N > 1:
    if N % cur == 0:
        N /= cur
        print(cur)
    else:
        cur += 1

{% endhighlight %}
